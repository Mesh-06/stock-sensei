# StockSense AI

AI-powered stock market forecasting and educational platform developed using Machine Learning, FastAPI, React, and Supabase.

## Features
- AI-based stock forecasting
- Real-time stock analysis
- Temporal Graph Transformer architecture
- Interactive frontend dashboard
- Educational AI chatbot
- Automated retraining pipeline

## Tech Stack
- Python
- PyTorch
- FastAPI
- React
- TypeScript
- Supabase
- Railway
- Vercel
